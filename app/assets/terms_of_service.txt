2. DESCRIPTION OF SERVICE
================================================================================

GezAI is an AI-powered travel planning mobile application that generates personalized walking, driving, and public transit routes for Turkish cities, currently focused on the Üsküdar district of Istanbul. The App provides the following key features:

- AI-generated route recommendations based on user preferences and input
- Integration with Google Maps for navigation and route visualization
- Access to place information including ratings, photos, and details
- Ability to save, bookmark, and manage generated routes
- Location-based services for route customization

The App utilizes artificial intelligence to process user input and generate route suggestions. Route information is enhanced using the Google Places API and displayed using Google Maps SDK.
